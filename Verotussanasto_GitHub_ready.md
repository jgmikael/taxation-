- **sv:** årsutgift

### Definition
- tulonhankkimiseen liittyvä meno, jonka <a href='http://uri.suomi.fi/terminology/verotus/c4' property ='related'>verovelvollinen</a> voi vähentää <a href='http://uri.suomi.fi/terminology/verotus/c3' property ='related'>verotuksessa</a> kokonaisuudessaan maksuvuonna

### Note
- Käsitekaaviot:<a href='http://www.tsk.fi/suomi.fi/verotussanasto/vero_vahennykset.svg' data-type='external'>Vähennykset</a> ja<a href='http://www.tsk.fi/suomi.fi/verotussanasto/vero_vuokratulot.svg' data-type='external'>Vuokratulosta vähennettävät menot</a>

## pay

<a id="pay"></a>

**IRI:** `https://iri.suomi.fi/terminology/verotus/c60`

### Labels

- **fi:** palkka
- **sv:** lön
- **en:** pay

### Definition
- palvelussuhteeseen, henkilökohtaiseen työpanokseen tai osallistumiseen perustuva vastike, josta sen maksajan on toimitettava <a href='http://uri.suomi.fi/terminology/verotus/c23' property ='related'>ennakonpidätys (1)</a> tai perittävä lähdevero ja maksettava sosiaalivakuutusmaksut

### Note
- Käsitekaavio:<a href='http://www.tsk.fi/suomi.fi/verotussanasto/vero_tulot.svg' data-type='external'>Tulot</a>
- Palkan ansainta- ja verovuosi voivat poiketa toisistaan. Palkka katsotaan sen <a href='http://uri.suomi.fi/terminology/verotus/c50' property ='related'>verovuoden</a> tuloksi, jona se on nostettu, merkitty <a href='http://uri.suomi.fi/terminology/verotus/c4' property ='related'>verovelvollisen</a> tilille tai muutoin saatu hallintaan.
- Palkkahallinnon näkökulmasta palkka määritellään hieman eri tavalla kuin verotuksessa. Palkkahallinnossa palkka rajataan palvelussuhteessa tai johtajasopimuksen perusteella tehdystä työstä maksettavaan tai maksettavaksi sovittuun vastikkeeseen.
- Verotuksen näkökulmasta palkaksi katsottavat suoritukset on lueteltu ennakkoperintälaissa (<a href='https://www.finlex.fi/fi/laki/ajantasa/1996/19961118' data-type='external'>1118/1996</a>).
- Palkansaajan <a href='http://uri.suomi.fi/terminology/verotus/c3' property ='related'>verotuksessa</a> palkka otetaan huomioon veronalaisena <a href='http://uri.suomi.fi/terminology/verotus/c54' property ='related'>ansiotulona</a>.
- Palkka maksetaan pääsääntöisesti rahana (rahapalkka), mutta se voidaan antaa myös <a href='http://uri.suomi.fi/terminology/verotus/c61' property ='narrower'>luontoisetuna</a>.

## will

<a id="will"></a>

**IRI:** `https://iri.suomi.fi/terminology/verotus/c174`

### Labels

- **fi:** testamentti
- **sv:** testamente
- **en:** will

### Definition
- henkilön määräys siitä, kenelle hänen omaisuutensa siirtyy hänen kuolemansa jälkeen

### Note
- Käsitekaavio:<a href='http://www.tsk.fi/suomi.fi/verotussanasto/vero_perintoverotus.svg' data-type='external'>Perintöverotus</a>
- Testamentti pitää ottaa <a href='http://uri.suomi.fi/terminology/verotus/c171' property ='related'>perinnönjaossa</a> huomioon. Testamentin vaikutus perinnönjakoon riippuu siitä, millainen testamentti on kyseessä. Erityistestamentissa (eli legaatissa) määrätään tietty esine tai yksilöity rahasumma testamentinsaajalle. Yleistestamentissa määrätään tietty osuus koko <a href='http://uri.suomi.fi/terminology/verotus/c162' property ='related'>jäämistöstä</a> testamentinsaajalle. Yleistestamentin saaja on <a href='http://uri.suomi.fi/terminology/verotus/c163' property ='related'>kuolinpesän</a> osakas, mutta erityistestamentin saaja ei ole.
- Testamentista säädetään perintökaaressa (<a href='https://www.finlex.fi/fi/laki/ajantasa/1965/19650040' data-type='external'>40/1965</a>).

## withholding

<a id="withholding"></a>

**IRI:** `https://iri.suomi.fi/terminology/verotus/c23`

### Labels

- **fi:** ennakonpidätys
- **sv:** förskottsinnehållning
- **en:** withholding

### Definition
- <a href='http://uri.suomi.fi/terminology/verotus/c19' property ='broader'>ennakkoperintä</a>, jossa tulon maksaja vähentää <a href='http://uri.suomi.fi/terminology/verotus/c4' property ='related'>verovelvolliselle</a> rahana maksettavasta tulosta <a href='http://uri.suomi.fi/terminology/verotus/c50' property ='related'>verovuodelle</a> arvioidun <a href='http://uri.suomi.fi/terminology/verotus/c1' property ='related'>veron</a> ja <a href='http://uri.suomi.fi/terminology/verotus/c2' property ='related'>veronluonteiset maksut</a> ja tilittää ne Verohallinnolle

### Note
- Käsitekaavio:<a href='http://www.tsk.fi/suomi.fi/verotussanasto/vero_ennakkoperinta.svg' data-type='external'>Ennakkoperintä</a>
- Ennakonpidätys (1) toimitetaan muun muassa <a href='http://uri.suomi.fi/terminology/verotus/c60' property ='related'>palkasta</a>, palkkioista ja eläkkeestä. Ennakonpidätys-termillä voidaan viitata myös ennakonpidätyksen määrään, ks. <a href='http://uri.suomi.fi/terminology/verotus/c26' property ='related'>ennakonpidätys (2)</a>.
- Ennakonpidätyksestä (1) säädetään ennakkoperintälaissa (<a href='https://www.finlex.fi/fi/laki/ajantasa/1996/19961118' data-type='external'>1118/1996</a>).

## withholding rate

<a id="withholding-rate"></a>

**IRI:** `https://iri.suomi.fi/terminology/verotus/c24`

### Labels

- **fi:** ennakonpidätysprosentti
- **sv:** förskottsinnehållningssats
- **en:** withholding rate

### Definition
- <a href='http://uri.suomi.fi/terminology/verotus/c102' property ='broader'>veroprosentti</a>, jonka mukaan <a href='http://uri.suomi.fi/terminology/verotus/c23' property ='related'>ennakonpidätys (1)</a> toimitetaan

### Note
- Käsitekaaviot:<a href='http://www.tsk.fi/suomi.fi/verotussanasto/vero_yleiset.svg' data-type='external'>Yleisiä käsitteitä</a> ja<a href='http://www.tsk.fi/suomi.fi/verotussanasto/vero_ennakkoperinta.svg' data-type='external'>Ennakkoperintä</a>
- Ennakonpidätysprosentti määritellään kaavamaisesti tai lasketaan arvioitujen tulojen ja <a href='http://uri.suomi.fi/terminology/verotus/c67' property ='related'>vähennysten</a> perusteella niin, että sen arvioidaan vastaavan <a href='http://uri.suomi.fi/terminology/verotus/c1' property ='related'>verojen</a> ja <a href='http://uri.suomi.fi/terminology/verotus/c2' property ='related'>veronluonteisten maksujen</a> osuutta <a href='http://uri.suomi.fi/terminology/verotus/c4' property ='related'>verovelvollisen</a> tuloista.
- Ennakonpidätysprosentti on merkitty verovelvollisen <a href='http://uri.suomi.fi/terminology/verotus/c27' property ='related'>verokorttiin</a> tai esitetty säädöksessä tai Verohallinnon päätöksessä.
- Ennakonpidätysprosentista säädetään ennakkoperintälaissa (<a href='https://www.finlex.fi/fi/laki/ajantasa/1996/19961118' data-type='external'>1118/1996</a>), ennakkoperintäasetuksessa (<a href='https://www.finlex.fi/fi/laki/ajantasa/1996/19961124' data-type='external'>1124/1996</a>) ja Verohallinnon vuosittain vahvistamassa päätöksessä ennakonpidätyksen toimittamistavoista ja määrästä.
